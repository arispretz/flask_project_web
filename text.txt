por unica vez en el nivel global:
pip install virtualenv

dentro del proyecto:
python -m venv venv

activar:
venv\Scripts\activate

instalar flask
pip install Flask

para listar los paquetes del proyecto:
pip freeze
copiar esa lista en requirements.txt y agregar gunicorn

para desactivar el entorno virtual:
deactivate

include the venv to your .gitignore file not to push it to github.

para mongodb"
pip install pymongo

para leer variables de entorno:
pip install environs

crear un nuevo repositorio en github

para insertar imagenes en visual studio code dos opciones:
1 descargar la imagen y desde downloads pasarlo a la carpeta en vsc
2 clic sobre la imagen guardar como directamente en el archivo ya creado en vsc

para conectar el proyecto con el repositorio de github:
git init
git add .
git commit -m "mensaje"
git remote add origin https://github.com/nombre_de_usuario/nombre_del_repositorio.git
git push -u origin master o main segun sea necesario
Debes usar git pull cuando quieres traer los cambios remotos a tu repositorio local.
git pull origin main
